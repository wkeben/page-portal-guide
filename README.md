# page-portal-guide

## 项目简介

这是一个用于归档和发布独立 HTML 页面的仓库。仓库内的页面不针对特定网站或域名，而是作为通用页面集合，方便浏览、索引与分享。

## 目录结构

```
page-portal-guide/
├── index.html          # 仓库首页 / 页面索引
├── pages/              # 存放各独立 HTML 页面
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
└── assets/             # 公共资源（样式、脚本、图片等）
    ├── style.css
    └── script.js
```

## 页面归档说明

- 所有页面均为独立的 HTML 文件，可直接在浏览器中打开或通过 GitHub Pages 访问。
- 页面不依赖特定后端服务，内容均为静态资源。
- 每个页面可附带简要说明（位于页面注释或目录中），以帮助了解用途。

## 使用方式

1. 克隆或下载本仓库。
2. 打开 `index.html` 查看页面索引。
3. 点击页面链接或直接访问 `pages/` 目录下的文件。

## 维护说明

- 欢迎提交 Pull Request 添加新的页面或改进现有页面。
- 页面应保持简洁、自包含，避免引入外部依赖（除非必需）。
- 提交前请确保页面在主流浏览器中可正常显示。
